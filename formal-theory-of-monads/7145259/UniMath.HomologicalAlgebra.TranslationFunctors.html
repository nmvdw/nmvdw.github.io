Translation funtor for C(A) and for K(A)

Introduction

We define the translation functor T : C(A) -> C(A) for complexes, and a translation functor T' : K(A) -> K(A). The functor T' is constructed so that we have the following commutative diagram C(A) ---T---> C(A) | | K(A) ---T'--> K(A) Here the vertical functors are given by ComplexHomotFunctor. The functor T sends a complex ... -> X^{i-1} --(d^{i-1})--> X^i --(d^i)--> X^{i+1} -> ... to the complex ... -> X^i --(-d^i)--> X^{i+1} --(-d^{i+1})--> X^{i+2} -> ... More precicely, on objects X^i ↦ X^{i+1} and differentials d^i_X ↦ -d^{i+1}_X . A morphism f : X -> Y is mapped by f^i ↦ f^{i+1} . We also construct the inverse translation T^{-1} which is the unique functor such that T ∘ T^{-1} = id and T^{-1} ∘ T = id. All the functors T, T^{-1}, and T' are additive.
The functor T : C(A) -> C(A) is constructed in TranslationFunctor. It is shown to be additive in TranslationFunctor_Additive. In TranslationFunctorPreservesHomotopies we show that T preserves homotopies, that is if f is homotopic to g, then T(f) is homotopic to T(g). In InvTranslationFunctor we construct T^{-1}.

  Local Lemma TranslationFunctor_comp (C : Complex A) (i : hz) :
    (to_inv (Diff C (i + 1))) · (to_inv (Diff C (i + 1 + 1))) =
    ZeroArrow (Additive.to_Zero A) (C (i + 1)) (C (i + 1 + 1 + 1)).
  Show proof.
    rewrite <- PreAdditive_invlcomp. rewrite <- PreAdditive_invrcomp.
    rewrite inv_inv_eq. apply (DSq A C (i + 1)).

  Definition TranslationComplex (C : Complex A) : Complex A.
  Show proof.
    use make_Complex.
    - intros i. exact (C (i + 1)).
    - intros i. exact (to_inv (Diff C (i + 1))).
    - intros i. exact (TranslationFunctor_comp C i).
  Global Transparent TranslationComplex.

  Definition DiffTranslationComplex (C : Complex A) (i : hz) :
